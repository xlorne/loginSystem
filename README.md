# C++ Hello World 示例工程

本项目是一个用于学习C++的工程实例，包含基础的用户登录系统实现，适合C++初学者参考和练习。

## 目录结构说明
- `src/`：源代码目录
- `include/`：头文件目录
- `CMakeLists.txt`：CMake构建脚本
- `conanfile.txt`：Conan依赖配置文件

## 主要内容
- 用户登录系统的简单实现
- C++类与对象的基本用法
- 使用CMake进行项目构建
- 使用Conan进行第三方库依赖管理（如fmt库）

## 依赖环境
- C++17 或更高版本编译器
- [CMake](https://cmake.org/) 3.15+
- [Conan](https://conan.io/) 2.x

## 快速开始
 
### 1. 构建项目

在项目根目录下执行：

```sh
sh build.sh
```

### 2. 运行程序

```sh
./build/login_system
```
